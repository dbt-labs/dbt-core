## 2.0.0-dev.18

Released July 17, 2026

### Fixes

- [dbt-fusion] Add +resource_tags config support for seeds and snapshots in dbt_project.yml
- [dbt-fusion] fix: explicitly set default empty variable for $SHELL so that the script will not end while setting the $SHELL.
- [dbt-core] Fix typo in the `try_or_compiler_error` type-check warning message ("a optional argument argument" -> "an optional argument") ([#15323](https://github.com/dbt-labs/dbt-core/issues/15323))
- [dbt-core] Fix source freshness failing entirely when a single source errors in the MapReduce aggregation phase. ([#15385](https://github.com/dbt-labs/dbt-core/issues/15385))
- [dbt-core] dbt init no longer leaves a stale authenticator field when switching from SSO/MFA to Password or Key pair authentication for Snowflake profiles ([#15378](https://github.com/dbt-labs/dbt-core/issues/15378))
- [dbt-core] Normalize BigQuery model context column data types when contract `alias_types` is enabled (e.g. `FLOAT` -> `FLOAT64`), matching dbt Core. ([#14097](https://github.com/dbt-labs/dbt-core/issues/14097))

### Under the Hood

- [dbt-core] Add ClickHouse to the list of non experimental adapters.
- [dbt-fusion] Publish pre-release versions to Homebrew and winget during the pre-stable window, and fix the empty-version bug in the release-v2 publish jobs.
